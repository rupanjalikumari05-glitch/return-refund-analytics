# 🛍️ Return & Refund Analytics (Fashion Retail)

## 📌 Project Overview

This project analyzes return and refund behavior in a fashion e-commerce business to identify key reasons for returns and reduce revenue loss.

---

## 🎯 Business Problem

Fashion e-commerce companies face high return rates due to:

* Size mismatch
* Delivery delays
* Product quality issues

---

## 🛠️ Tools Used

* MySQL
* Python (Pandas, Matplotlib, Seaborn)
* Power BI
* Excel

---

## 📊 Key Insights

* Highest Return Reason: Late Delivery
* Most Returned Category: Dress
* Revenue Lost: ₹19,69,787
